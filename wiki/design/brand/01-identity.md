# 01. Brand Identity

> **Owner**: 김성곤 · **Status**: In Review · **Last Updated**: 2026-04-26 · **Source**: Brand Identity Guide v1.1

브랜드의 미션, 포지셔닝, 트레잇, 톤앤매너를 정의한다. 이 문서는 모든 디자인 결정의 상위 기준이다.

---

## 1. 브랜드 한 줄

> **Bite-sized learning, on auto-play** — 한 입 크기로, 자동으로 재생되는 학습.

## 2. 미션 / 포지셔닝

- **미션**: 긴 강의·교과서를 60초 숏폼으로 자동 편집해, 매일 한 입씩 즐겁게 학습하게 한다.
- **포지셔닝**: 듀오링고가 _언어 학습의 매일매일_ 을 캐릭터로 풀었다면, Shortify는 _60초짜리 호기심 한 입_ 을 다룬다.
- **차별 축**: **숏폼 · AI · macOS 데스크탑 앱** — 듀오링고의 따뜻한 게임감은 가져가되, 시각적으로 뚜렷이 다른 자리에 선다.

## 3. 브랜드 트레잇 (Traits)

| 키워드 | 의미 | 디자인 표현 |
|--------|------|--------------|
| **Snappy** | 짧고 강하게 | 60초 단위, 한 카드 한 개념, 빠른 트랜지션 |
| **Friendly** | 친구처럼 가깝게 | 듀오링고 톤. 전문가가 아닌 옆자리 친구의 말투 |
| **Smart** | 똑똑하게 핵심만 | AI가 정수만 뽑아주는 신뢰감 |

## 4. 톤앤매너 (Voice & Tone)

듀오링고의 비밀은 카피라이팅에 있다 — Shortify도 같은 원칙으로 운영한다.

### 4.1 원칙

1. **존댓말 기본, 단 문장은 짧게** — "3분이면 끝나요", "한 입만 더 어때요?"
2. **숫자와 의성어를 즐겨** — "60초", "한 입", "오늘 14일 연속!"
3. **실패에 관대, 성공엔 호들갑** — 틀려도 "괜찮아요, 다시 해볼까요?", 맞으면 "와! 완벽해요"
4. **전문 용어는 풀어서** — "Vector embedding" → "AI가 비슷한 것끼리 묶어주는 방법"

### 4.2 카피 예시

| 상황 | OK | NG |
|------|----|----|
| 환영 | 와, 오셨네요! 오늘 한 입 시작해볼까요? | 안녕하세요. 학습을 시작하시려면 아래 버튼을 클릭하세요. |
| 정답 | 정확해요! 척척이에요 | 정답입니다. |
| 오답 | 거의 다 왔어요. 한 번만 더! | 오답입니다. 다시 시도해주세요. |
| 스트릭 알림 | 14일 연속! 멈추기엔 너무 멋져요 | 14일 연속 학습 중입니다. 오늘도 학습해주세요. |

### 4.3 톤 좌표

| 차원 | 위치 |
|------|------|
| 격식 ◀────●─▶ 친근 | 친근 쪽으로 8/10 — 옆자리 친구 톤. |
| 진지 ◀───●──▶ 유쾌 | 유쾌 6/10 — 단, 학습 신뢰는 잃지 않게. |
| 정적 ◀──●───▶ 동적 | 동적 6/10 — 마이크로 인터랙션 위주, 산만 금지. |
| 미니멀 ◀──●───▶ 풍부 | 풍부 6/10 — 캐릭터 · 컬러로 따뜻함. |

## 5. 페르소나

- **핵심 사용자**: 퇴근 후 30분, 새 개념 하나를 가볍게 익히고 싶은 직장인 / 자기학습 학습자.
- **사용 맥락**: 책상 앞 macOS — 긴 강의 영상이나 PDF를 두고 "이거 60초로 줄여줄 친구"가 필요한 순간.
- **감정 약속**: "오늘도 한 입은 했네" — 부담 없는 매일의 성취.

## 6. 안 하는 것 (Negative Space)

- ❌ 압박형 카피 ("공부 안 하면 슬퍼져요" 같은 길티 트리거).
- ❌ 에듀테크 클리셰 그린/블루를 메인으로 쓰는 것.
- ❌ 캐릭터를 학습 영상 본편 안에 넣는 것 (출처/콘텐츠 신뢰성 우선).
- ❌ 순흑 `#000` / 순백 `#FFF` 사용 (Ink/Cream으로 통일).
- ❌ 격식체·관공서 톤 (옆자리 친구 톤 유지).

## 7. 레퍼런스 대비 차별 포인트

| | Duolingo | Shortify |
|---|---|---|
| 메인 컬러 | 그린 `#58CC02` | 코랄 `#FF6B4A` |
| 핵심 단위 | 레슨 1개 | 60초 숏폼 1개 |
| 마스코트 성격 | 부담 주는 부엉이 (Duo) | 한 입 더 권하는 레서판다 (쇼리) |
| 주요 플랫폼 | 모바일 우선 | macOS 데스크탑 우선 |
| 게이미피케이션 | XP · 리그 · 하트 | XP · 스트릭 · 컬렉션 (영상 모음) |
| 콘텐츠 생성 | 사람이 큐레이션 | AI 자동 생성 |
