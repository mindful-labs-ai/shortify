# 01. Character Bible — 쇼리 (Shori)

> **Owner**: 김성곤 · **Status**: Approved · **Last Updated**: 2026-04-26 · **Source**: Brand Identity Guide v1.1 + Character Sheet v1

Shortify 마스코트 **쇼리(Shori)** 의 성격, 비주얼, 세계관을 정의한다. 사용 규칙은 [02-usage](./02-usage.md).

> 듀오링고에서 가장 중요한 자산이 Duo이듯, Shortify도 **마스코트를 핵심 자산** 으로 가져간다. 광고 · 알림 · 빈 화면 · 축하 모먼트 등 사용자가 감정을 느끼는 거의 모든 접점에 등장시킨다.

## 1. 이름 / 정체

| 항목 | 내용 |
|------|------|
| **이름** | 쇼리 (Shori) |
| **종(Species)** | 레서판다 (Red panda) |
| **역할 in 앱** | "한 입만 더 어때요?" 라고 권유하는 옆자리 친구. 진행을 안내하고, 빈 라이브러리에서 인사하고, 보상 모먼트에 폭죽을 든다. |

### 네이밍 로직

- "**shor**tify"의 앞부분에서 따왔다 — 캐릭터 ↔ 브랜드 자연스러운 연결.
- 한국어/영문 표기(Shori) 모두 같은 발음 → 글로벌 확장 시 안전.
- "쇼리야~", "쇼리쇼리" 처럼 의성어로 변주 가능 → IP 확장에 유리.

## 2. 성격 / 메시징 4 필러

캐릭터 시트(v1)에서 확정된 4개 메시지 필러:

| 필러 | 한 줄 |
|------|------|
| 한 입만 더 권하는 친구 | 부담 없이 매일 한 입을 권유하는 옆자리 친구. |
| 빠르고 간편해요 | 60초 한 입 — Snappy. |
| 똑똑하게 핵심만 쏙쏙 | AI가 정수만 뽑아주는 신뢰감. |
| 함께 배우면 더 즐거워요 | 학습은 외롭지 않게, 곁에 있는 동반자. |

### 톤 키워드 (3 Words)

- **호기심** — 새 PDF 드롭에 눈이 반짝.
- **친근함** — 부담을 주지 않는다. 권유형 톤.
- **꾸준함** — 매일 한 입씩, 조용히 옆에 있는 친구.

> 핵심 메시지: **"한 입 더 먹자!"** (Duo가 "공부 안 하면 슬퍼져"라면, 쇼리는 "한 입만 더 어때요?")

## 3. 콘셉트 매칭 (왜 레서판다인가)

레서판다는 _대나무를 조금씩, 자주 먹는_ 동물이다. **"한 입씩 자주" 학습** 한다는 Shortify의 핵심 메타포와 정확히 일치한다.

추가 강점:
- 둥글둥글한 비율 → 친근함.
- 줄무늬 꼬리 → 시각 시그니처(아이덴티티 식별 요소).
- 부엉이/펭귄 등 에듀테크 마스코트와 겹치지 않음.

## 4. 비주얼 명세

| 요소 | 결정 | 비고 |
|------|------|------|
| 스타일 | 플랫 일러스트, 단순 셀쉐이딩 | 산만한 그라디언트 금지 |
| 비율 | 둥글고 통통 — 큰 머리, 짧은 팔다리 | head:body ≈ 1:1 ~ 1:1.2 |
| 본체 컬러 | Spark coral `#FF6B4A` | 브랜드 메인과 동일 |
| 페이스 마스크 | Cream `#FFF8F2` | 눈 주변, 주둥이, 가슴, 귀 안쪽 |
| 포인트 컬러 | Dark Brown `#3A2A23` | 눈, 귀 끝, 발, 코 — 시각적 무게중심 (캐릭터 시트 v1에서 확정) |
| 블러쉬 | 옅은 핑크 볼터치 | 친근함 강조 (HEX 후보 `#FFC1B3` — 다음 토큰 작업에서 확정) |
| 꼬리 | Spark coral + 어두운 코랄(`coral-700`) 줄무늬 | 레서판다 트레이드마크, 식별 요소 |
| 라인 | 베이스는 outline 없음 — 컬러 면 분리로 형태감을 만듦. 강조가 필요한 경우만 Dark Brown 1.5~2px | 캐릭터 시트 v1 기준 |
| 음영 | 단색 평면 / 단순 셀쉐이딩 (얼굴 마스크 안쪽에 옅은 그림자 1톤만) | 그라디언트 · 3D · 베벨 금지 |

> 컬러 코드 매핑: [03-color](../brand/03-color.md). 캐릭터 시트 v1에서 Dark Brown은 `#3A2A23` 으로 확정 — [03-color §1](../brand/03-color.md#1-코어-팔레트) 토큰 추가 항목.

## 5. 세계관 / 백스토리 (간단)

> 쇼리는 사용자의 책상 어딘가에 사는 작은 레서판다다. 매일 새 PDF가 도착하면 코를 킁킁대며 살펴보고, 한 입 베어 물기 좋은 크기로 잘라 가져다 준다. 시끄럽지 않다 — 공부에 방해되지 않게, 옆에서 조용히 응원한다.

## 6. 표정 / 포즈 / Turnaround

캐릭터 시트 v1에서 8 표정 + 4 포즈 + 4-view turnaround + 5 사용 예가 확정되었다. 모든 자산은 [`/design/character/final/sheet/character_mascot_sheet_v1_final.png`](../../../design/character/final/sheet/character_mascot_sheet_v1_final.png) 한 장에 들어 있다.

### 6.1 표정 셋 (Expressions, 8종)

| 키 | 시트 라벨 | 톤 | 사용 위치 |
|----|----------|----|-----------|
| `default` | 기본 | 가벼운 미소 | 기본 idle |
| `excited` | 신나요! | 눈 반짝, 입 활짝 | 새 PDF 드롭, 시작 모먼트 |
| `curious` | 궁금해요 | 발 짚은 채 갸웃 | 가이드, 빈 상태 |
| `surprise` | 깜짝! | 눈 동그랗게, 입 작게 벌림 | 액션 피드백 (drop 직후) |
| `proud` | 뿌듯해요 | 만족스러운 눈웃음 | 완료, 보상 직후 |
| `disappointed` | 아쉬워요 | 눈 살짝 내림 | 오답, 실패 모먼트 |
| `sleepy` | 졸려요 (Z) | 눈 감고 표정 부드럽게 | 빈 라이브러리, 휴면 시간대 |
| `cheer` | 화이팅! | 양손에 폼폼 든 응원 | 스트릭 격려, CTA 직전 |

### 6.2 포즈 셋 (Poses & Actions, 4종)

| 키 | 시트 라벨 | 사용 위치 |
|----|----------|-----------|
| `study-start` | 공부 시작! | 학습 시작 — 책 펼치고 앉음 |
| `bite-more` | 한 입만 더! | 진행 중간 — "한 입 더 권유" 카피와 매칭 |
| `streak-flag` | 연속 학습 달성! | 스트릭 갱신 — 깃발 들고 점프 |
| `take-break` | 잠깐 쉬어가요 | 휴식 안내 — 컵을 들고 앉음 |

### 6.3 4-view Turnaround

캐릭터 시트의 CHARACTER TURNAROUND 행:

| 키 | 라벨 | 용도 |
|----|------|------|
| `front` | FRONT | 개발 인계 정본 (단일 컷: `/design/character/final/turnaround/character_mascot_front_v1_final.png`) |
| `three-quarter` | 3/4 SIDE | 일러스트/마케팅 일반 |
| `side` | SIDE | 모션 가이드 / 측면 모먼트 |
| `back` | BACK | 꼬리 강조용 — 스트릭/컬렉션 장면 |

### 6.4 IN USE 시나리오 (시트 v1 확정)

캐릭터 시트의 IN USE 행은 앱 내 5개 핵심 모먼트 + 카피와 일대일 매칭된다 — 사용 가이드는 [02-usage §1](./02-usage.md#1-사용-시나리오-앱-내).

| 시나리오 | 카피 | 표정/포즈 |
|----------|------|-----------|
| 데스크에서 학습 시작 | "오늘의 한 입 시작!" | `study-start` + `default` |
| 정답 모먼트 | "정답! 잘했어요!" | `proud` (박수 + 폭죽) |
| 14일 스트릭 | "14일 연속 달성!" | `streak-flag` + 캘린더 |
| 호기심 유도 | "무슨 내용일까?" | `curious` + 말풍선 |
| 야간/취침 알림 | "잘 자요, 내일 또 만나요!" | `sleepy` + 이불 |

## 7. 5인 캐스트와의 관계 (앱 인격 vs 콘텐츠 인격)

쇼리는 **앱의 인격(브랜드 마스코트)**, 영상 안에서 지식을 전달하는 인간 캐스트는 **콘텐츠의 인격(화자)**. 둘은 등장 위치가 분리된다.

| 구분 | 쇼리 | 5인 캐스트 (Pip · Iris · Jay · Vera · Sage) |
|------|------|----------------------------------------------|
| 등장 위치 | 푸시 / 빈 화면 / 스트릭 / 앱 UI 본편 | **영상 안** (내레이터 / 진행자 / 시연자) |
| 영상 본편 안 | ❌ ([02-usage §2](./02-usage.md#2-사용-위치-앱-외)) | ✅ |
| 컬러 | Spark coral `#FF6B4A` (메인) | 보조 팔레트 (yellow / mint / sky / lavender / warm-gray) |
| 동시 등장 | 영상 모서리에서 박수치는 등 보조 등장 가능 — 캐스트와 컬러가 충돌하지 않도록 설계됨 | 한 영상에 1~2인 |

상세 캐스트 정의는 [03-cast](./03-cast.md).

## 8. 듀오링고 Duo와의 차별

| | Duo (Duolingo) | 쇼리 (Shortify) |
|---|---|---|
| 동물 | 부엉이 | 레서판다 |
| 핵심 메시지 | "공부 안 하면 슬퍼져" | "한 입 더 먹자!" |
| 톤 | 압박형 | 권유형 |
| 메타포 | 지혜의 상징 | 자주 조금씩 먹는 동물 = 한 입 학습 |
| 시각 시그니처 | 큰 눈 + 노려보는 표정 | 줄무늬 꼬리 + 페이스 마스크 |
