## 산출물 셋

확정 캐릭터(`final/`)에는 최소한 다음을 둔다:
- **base**: 정면 / 측면 / 후면 turnaround
- **expressions**: 기본 · 기쁨 · 집중 · 놀람 · (학습 톤에 맞는 차분한 표정 위주)
- **poses**: standing · pointing · holding-pdf · waving 등 앱 내 사용 포즈
- **scaled exports**: 24px / 64px / 256px / 1024px PNG (transparent)

상세 사용 규칙은 [`02-usage`](../../wiki/design/character/02-usage.md) 참고.
