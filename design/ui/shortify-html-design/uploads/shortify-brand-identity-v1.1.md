# shortify Brand Identity Guide

> **버전** v1.1 · 마스코트 확정
> **서비스** macOS용 교육 숏폼 AI 자동 제작 데스크탑 앱
> **레퍼런스** Duolingo

---

## 1. 브랜드 컨셉

### 한 줄 정의
**Bite-sized learning, on auto-play** — 한 입 크기로, 자동으로 재생되는 학습.

### 포지셔닝
듀오링고가 _언어 학습의 매일매일_ 을 캐릭터로 풀었다면, shortify는 _60초짜리 호기심 한 입_ 이라는 포지션에서 출발합니다. 친근하고 가볍고 게임 같은 무드는 가져가되, **숏폼 · AI · macOS 데스크탑 앱** 이라는 차이를 시각적으로 뚜렷하게 만듭니다.

### 브랜드 트레잇

| 키워드 | 의미 | 디자인 표현 |
| --- | --- | --- |
| Snappy | 짧고 강하게 | 60초 단위, 한 카드 한 개념, 빠른 트랜지션 |
| Friendly | 친구처럼 가깝게 | 듀오링고 톤. 전문가가 아닌 옆자리 친구의 말투 |
| Smart | 똑똑하게 핵심만 | AI가 정수만 뽑아주는 신뢰감 |

## 2. 컬러 시스템

에듀테크에 흔한 그린 · 블루를 피하고 **코랄을 메인** 으로 잡았습니다. 듀오링고 그린이 가진 채도감과 따뜻함은 유지하면서, 시장에서의 식별성을 확보하기 위한 선택입니다. 동시에 코랄은 "재생 · 플레이 버튼"의 상징색으로도 자연스러워 숏폼 서비스와 결이 맞습니다.

### 코어 팔레트

| 역할 | 이름 | HEX | RGB | 사용처 |
| --- | --- | --- | --- | --- |
| Primary | Spark coral | `#FF6B4A` | 255, 107, 74 | 로고, 주요 CTA, 브랜드 |
| Primary dark | Ember | `#E04A2B` | 224, 74, 43 | hover / active 상태 |
| Accent | Sunny yellow | `#FFC83D` | 255, 200, 61 | 스트릭, 보상, 축하 모먼트 |
| Success | Mint green | `#5BD4A8` | 91, 212, 168 | 정답, 완료, 진척도 |
| Info | Sky blue | `#4A9BFF` | 74, 155, 255 | 링크, 힌트, 정보 |
| Text | Ink | `#1A1614` | 26, 22, 20 | 본문, 헤딩 (순흑 대신 살짝 따뜻) |
| Surface | Cream | `#FFF8F2` | 255, 248, 242 | 메인 배경 (순백 대신 따뜻한 오프화이트) |
| Subtle bg | Cloud | `#F5EFE8` | 245, 239, 232 | 카드 위계용 옅은 배경 |

### 보조 셰이드 (UI 빌드용)

```
Spark coral
  50  #FFF1EB    배경 박스
  100 #FFD9C9    호버 배경
  500 #FF6B4A    primary
  700 #E04A2B    primary dark
  900 #993C1D    coral 위 텍스트

Sunny yellow
  50  #FFF4D6
  500 #FFC83D
  900 #7A5A00

Mint green
  50  #E0F7EE
  500 #5BD4A8
  900 #0F6E56
```

### 컬러 운영 원칙

1. **메인 코랄은 한 화면에 한 영역만** — 사용자의 눈이 가장 중요한 액션으로 향하도록.
2. **옐로우는 사용자 칭찬의 순간에만** — 스트릭 카운터, 별, 폭죽 애니메이션 등에 한정해 의미를 보존합니다.
3. **민트는 정답 · 체크마크 전용** — 묶어 두면 듀오링고 그린이 주는 "맞았다!"의 쾌감을 변주할 수 있습니다.
4. **순흑 · 순백은 사용하지 않습니다** — Ink와 Cream으로 톤을 통일해 전체 인상이 따뜻하게 유지됩니다.

## 3. 타이포그래피

| 용도 | 한국어 | 영문 | 비고 |
| --- | --- | --- | --- |
| Display (헤더, 웰컴 화면) | Pretendard ExtraBold (800) | Nunito ExtraBold | 둥글고 두꺼운 인상으로 듀오링고 'Feather Bold' 분위기 차용 |
| Heading | Pretendard SemiBold (600) | Nunito Bold | 카드 제목, 섹션 헤더 |
| Body | Pretendard Regular / Medium (400 / 500) | Inter 400 / 500 | 가독성 우선 |
| Numeric · Mono | JetBrains Mono | 동일 | 카운터, 진척도, 코드성 정보 |

### 사이즈 스케일 (macOS 기준)

```
Display L   48 / 56  -2% letter-spacing
Display M   36 / 44  -2%
H1          28 / 36  -1%
H2          22 / 30  -0.5%
H3          18 / 26
Body L      16 / 26
Body M      14 / 22
Caption     12 / 18
Mono S      13 / 20
```

### 샘플 카피

> **Display** — 오늘의 한 입, 3분이면 충분
>
> **Body** — 긴 강의를 AI가 60초 숏폼으로 자동 편집해주는 맥용 데스크탑 앱이에요. 짧게, 자주, 즐겁게.
>
> **Mono** — `streak · 14d  |  level · 03  |  xp · 1,240`

## 4. 로고와 앱 아이콘

### 워드마크

소문자 `shortify`. 듀오링고처럼 친근하고 비격식적인 인상. Pretendard ExtraBold (또는 Nunito ExtraBold) 기반에 자간 -2~-4% 정도로 단단하게 묶습니다.

**최소 크기**
- 디지털 24px 이상
- 인쇄 8mm 이상

**클리어 스페이스** — 워드마크 높이의 0.5배를 사방에 확보.

### 앱 아이콘 (macOS, 1024 × 1024)

- **외형** 22px 라운드 스퀘어 (Sonoma 가이드라인 준수)
- **배경** 단색 Spark coral `#FF6B4A`
- **마크** 가운데 Cream 색 `#FFF8F2` 플레이 삼각형
- **컨셉** "S"의 곡선과 플레이 버튼을 하나의 형태로 묶어, 서비스 이름의 첫 글자이자 동시에 "재생"의 상징이 되도록 설계
- macOS Sonoma 이후의 평면적이고 명확한 아이콘 트렌드와도 일치

## 5. 마스코트 — 쇼리 (Shori)

듀오링고에서 가장 중요한 자산이 Duo이듯, shortify도 **마스코트를 핵심 자산** 으로 가져가는 것을 권장합니다. 광고, 알림, 빈 화면, 축하 모먼트 등 사용자가 감정을 느끼는 거의 모든 접점에 등장시킵니다.

### 네이밍 로직

**쇼리** 는 "**shor**tify"의 앞부분에서 따왔습니다. 사용자가 캐릭터를 보는 순간 자연스럽게 브랜드와 연결되도록 설계했고, 한국어로도 영문 표기(Shori)로도 같은 발음이라 글로벌 확장 시에도 안전합니다. "쇼리야~", "쇼리쇼리" 처럼 의성어로 변주하기 좋아 IP로 키우기 유리한 이름이기도 합니다.

### 캐릭터 설정

| 항목 | 내용 |
| --- | --- |
| 이름 | 쇼리 (Shori) |
| 종족 | 레서판다 (Red panda) |
| 컬러 | Spark coral 본체 + Cream 페이스 마스크 + 다크 브라운 포인트 |
| 형태 | 둥글고 통통한 비율, 큰 머리와 짧은 팔다리. 줄무늬 꼬리가 시그니처 |
| 콘셉트 매칭 | 레서판다는 _대나무를 조금씩 자주 먹는_ 동물. "한 입씩 자주" 학습하는 shortify의 핵심 메타포와 정확히 일치 |
| 성격 | "한 입만 더 어때요?" 라고 조르는 친구. 부담 없이 즐겁게 학습을 권유 |

### 시각 디테일

- **본체 컬러** Spark coral `#FF6B4A` — 브랜드 메인 컬러와 동일
- **페이스 마스크** Cream `#FFF8F2` (눈 주변, 주둥이, 가슴, 귀 안쪽)
- **포인트 컬러** 다크 브라운 (눈, 귀 끝, 발, 코) — 시각적 무게중심
- **블러쉬** 옅은 핑크 볼터치 — 친근함 강조
- **꼬리** 코랄 + 어두운 코랄 줄무늬 — 레서판다의 트레이드마크이자 아이덴티티 식별 요소

### 활용 시나리오

- **온보딩** — 손 흔들며 인사하는 쇼리
- **빈 화면** — 꼬리에 몸을 말고 잠든 쇼리 + "오늘은 뭐 배워볼까요?"
- **학습 시작** — 대나무(=콘텐츠) 한 입 베어 무는 쇼리
- **정답** — 박수치는 쇼리
- **연속 학습 달성** — 폭죽 들고 점프하는 쇼리, 꼬리가 위로 올라간 자세
- **휴면 사용자** — 살짝 시무룩한 쇼리 + 푸시 알림 ("쇼리가 기다려요")

### 듀오링고 Duo와의 차별

| | Duo (듀오링고) | 쇼리 (shortify) |
| --- | --- | --- |
| 동물 | 부엉이 | 레서판다 |
| 핵심 메시지 | "공부 안 하면 슬퍼져" | "한 입 더 먹자!" |
| 톤 | 압박형 | 권유형 |
| 메타포 | 지혜의 상징 | 자주 조금씩 먹는 동물 = 한 입 학습 |
| 시각 시그니처 | 큰 눈 + 노려보는 표정 | 줄무늬 꼬리 + 페이스 마스크 |

## 6. 보이스 & 톤

듀오링고의 비밀은 사실 카피라이팅에 있습니다. shortify도 같은 원칙으로 운영합니다.

### 원칙

1. **존댓말 기본, 단 문장은 짧게** — "3분이면 끝나요", "한 입만 더 어때요?"
2. **숫자와 의성어를 즐겨** — "60초", "한 입", "오늘 14일 연속!"
3. **실패에 관대, 성공엔 호들갑** — 틀려도 "괜찮아요, 다시 해볼까요?", 맞으면 "와! 완벽해요"
4. **전문 용어는 풀어서** — "Vector embedding" 대신 "AI가 비슷한 것끼리 묶어주는 방법"

### 카피 예시

| 상황 | OK | NG |
| --- | --- | --- |
| 환영 | 와, 오셨네요! 오늘 한 입 시작해볼까요? | 안녕하세요. 학습을 시작하시려면 아래 버튼을 클릭하세요. |
| 정답 | 정확해요! 척척이에요 | 정답입니다. |
| 오답 | 거의 다 왔어요. 한 번만 더! | 오답입니다. 다시 시도해주세요. |
| 스트릭 알림 | 14일 연속! 멈추기엔 너무 멋져요 | 14일 연속 학습 중입니다. 오늘도 학습해주세요. |
